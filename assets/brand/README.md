# Your Energy Logo Pack

This folder contains the website-ready logo files for `Your Energy`.

Recommended usage:

- `../logo-your-energy.svg`
  Use as the default full-color logo on light backgrounds.

- `your-energy-logo-reverse.svg`
  Use on dark backgrounds, hero panels, and overlays.

- `your-energy-logo-monochrome.svg`
  Use for one-color applications, print mocks, or emboss-style branding.

- `../your-energy-mark.svg`
  Use for compact placements such as sticky headers, app-style badges, social avatars, and small UI surfaces.

- `your-energy-mark-reverse.svg`
  Use for compact placements on dark backgrounds.

- `../favicon.svg`
  Use as the main modern favicon source.
